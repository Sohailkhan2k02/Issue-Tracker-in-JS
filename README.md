A simple plugin which can be used to keep track of issues. You can just copy pase the source code and use it easily as per your need with
desired changes.

<h3>It keeps track of the following : </h3>
<ul>
<li>Description of the issue</li>

<li>Severity of the issue</li>

<li>Whom is the issue assigned</li>
</ul>
